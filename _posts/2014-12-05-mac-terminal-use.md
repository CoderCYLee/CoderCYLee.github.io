## 一些快捷键

```
crtl + l // 清屏
ctrl + a // 移到行首
ctrl + e // 移到行尾
ctrl + y // 插入最近删除的单词或语句
ctrl + k // 删除光标处到行尾部分
ctrl + u // 删除光标处到行首部分
ctrl + w // 删除光标处到当前单词开头部分或语句
```

```
date // 显示当时日期
time // 显示当前时间
cal  // 显示日历
```
